MDPs

In this assignment, you'll implement value iteration and policy iteration as per the slide deck covered in class. Note that in contrast to the code from the 4th edition of the book, here, rewards are defined over states (not over state, action, successor-state triples). Further, the utility of a particular state includes the immediate reward obtained in that state.

Command Line Invocation

Value and Policy iteration should be implemented as top-level functions. These functions will be called directly from the test harness. However, a "main" method has also been provided that may help you test your code on your own.

Usage instructions are given when you invoke the function at the command line with no arguments:

python mdp.py usage: mdp.py [-h] [--quiet] [--gamma GAMMA] {policy,value} {4x3,2x2} mdp.py: error: too few arguments

the first required argument selects the function to run, the second identifies which of the two provided environments to use.

python msttt.py policy 4x3

for example, runs policy iteration on the 4x3 environment
